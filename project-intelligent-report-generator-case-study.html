Background + Problem

What needed to be solved

Challenge

Raw reviews and product images were difficult to convert into compact, useful, and trustworthy recommendations. Existing outputs were either too verbose, too generic, or failed to reconcile visual and textual signals.

Requirements

  • Fuse image and text understanding in one pipeline.
  • Extract sentiment, emotions, and product aspects.
  • Produce clean, structured reports for end users.
  • Expose model outputs through a practical interface.
Technical Solution

System architecture and model stack

Pipeline Workflow

Complete orchestration from image and text ingestion to final structured report generation.

Vision Layer

EfficientNetB3 identified product-level visual features and categories from image inputs, creating normalized visual embeddings.

NLP Layer

Parallel stack with BiLSTM, BERT, and SmolLM2 handled sentiment, emotional tone, and feature extraction from user reviews.
